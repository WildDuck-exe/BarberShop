# BarberShop
Projeto Java para gerenciamento de barbearia, desenvolvido no NetBeans.

## Como rodar
- Abra o projeto no NetBeans.
- Clique em Run > Run Project (F6).

## Estrutura
- `src/`: Código-fonte (Model, View, etc.).
- `manifest.mf`: Configurações do JAR.
